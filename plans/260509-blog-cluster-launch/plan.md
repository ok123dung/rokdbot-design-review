## Lessons Learned

### Iteration is mandatory
7 vòng loop để đến đúng tone:
1. Generic feature article → 2. PAS structure → 3. Case study angle → 4. Pricing tier accurate → 5. Per-day breakdown → 6. Victory metric clarification → 7. Per-feature split + cross-link

### Don't over-research before showing
First article (Auto Rally) wrote with too many TODO placeholders. Better: write text version → user feedback → swap images.

### Vietnamese-native terms matter
"kéo rợ", "đạo", "rợ" tốt cho SEO + relatable. "barb chain", "march" generic.

### Bot scope precision
User clarified V2/V3/Premium scope mid-session. Important to ASK explicitly which tier owns which feature before writing.
